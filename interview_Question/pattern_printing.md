#Print 1 To 100 
```python
for i in range(1,101):
    print(i)
```

```Python
n = 100
i = 1
while i <= n:
    print(i)
    i = i +1
```
#Printing 20 to 1
```Python
i = 20
while i > 0:
    print(i)
    i = i - 1
```
```Python
i = 1
n = 20
while i <= n:
    print(i)

```
#Printing even numbers from 1 to 20
```Python
i = 1
while i <= 20:
    if i % 2 ==0:
        print(i)
    i = i+1
```
```Python
i = 0
while i <= 20:
    print(i)
    i = i + 2
```
