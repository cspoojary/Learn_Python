### Swap Two Variables
```python
a = 5
b = 6

a, b = b, a
print('a=',a,'and b =',b)

a = a + b   # a = 11
b = a - b   # b = 5
a = a - b   # a = 6

print('a=',a,'and b =',b)
```

### Type Conversion
```python
str1 = "123"
print(type(str1))
intgr = int(str1)
print(type(intgr))
flt = float(str1)
print(type(flt))
```
